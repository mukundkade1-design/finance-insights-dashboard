# 📊 Financial Reporting and Analysis Dashboard (Power BI)

This project is a Financial Reporting and Analysis Dashboard built using Microsoft Power BI. It helps in analyzing financial data and generating meaningful business insights.

---

## 📁 Files Included
- Finanicial_Reporting_And_Analysis(PowerBI).pbix → Main Power BI dashboard file  
- Screenshot 2026-07-05 131711.png → Dashboard overview  
- Screenshot 2026-07-05 131930.png → Detailed analysis view  

---

## 🎯 Objective
- Visualize financial performance  
- Analyze revenue, profit, and expenses  
- Understand business trends  
- Support data-driven decision making  

---

## 📊 Dashboard Features
- Revenue & Profit Analysis  
- Expense Tracking  
- Key Performance Indicators (KPIs)  
- Monthly / Yearly Trend Analysis  
- Interactive filters and slicers  

---

## 🛠 Tools Used
- Microsoft Power BI  
- Excel / CSV dataset  
- Power Query for data cleaning and transformation
